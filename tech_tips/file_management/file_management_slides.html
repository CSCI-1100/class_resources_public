Why these work:

  • Course identifier first
  • Content type specified
  • Version/date included
  • Your name for submissions

Problems:

  • Meaningless names
  • No course identification
  • No version control
  • Impossible to search

Troubleshooting Common Issues

OneDrive Not Syncing

  • Check internet connection
  • Right-click OneDrive icon ‚Üí Settings
  • Click "Reset OneDrive" if problems persist
  • Sign out and sign back in

File Conflicts

  • Look for "Conflicted Copy" in filename
  • Compare versions manually
  • Merge changes if needed
  • Delete conflicted copy after merging

Can't Find Files

  • Use OneDrive web interface (onedrive.com)
  • Check Recycle Bin (both local and OneDrive)
  • Search by filename in File Explorer
  • Check Version History for content changes

Running Out of Space

  • Delete old/unnecessary files
  • Use "Files On-Demand" (cloud-only storage)
  • Empty OneDrive Recycle Bin
  • Check university storage limits

Best Practices Checklist

Organization

  • Create consistent folder structures
  • Use descriptive, standardized file names
  • Organize by course, then content type
  • Keep active and archived materials separate

Collaboration

  • Share folders, not individual files when possible
  • Establish naming conventions with team members
  • Use comments and suggestions for feedback
  • Regularly sync to avoid conflicts

Backup and Security

  • Don't rely solely on OneDrive - keep local backups of critical files
  • Download final versions of important assignments
  • Be cautious with sharing permissions
  • Regularly review and clean up shared folders
