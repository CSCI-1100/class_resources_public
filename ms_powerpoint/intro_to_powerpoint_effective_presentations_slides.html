Getting Started with PowerPoint

File Management & Auto-Save

Creating a New Presentation

  1. Go to File → New
  2. Choose "Blank Presentation" or select a template
  3. Save immediately to enable auto-save
Pro Tip: Save to OneDrive or cloud storage to enable auto-save (look for the circular arrow icon)

File Formats

  • .pptx - Standard PowerPoint format
  • Compatible with Google Slides
  • Open format for cross-platform use
Demo: Create new presentation and show auto-save feature

Templates and Themes

Choosing the Right Starting Point

Built-in Templates

  • Home tab: Basic template gallery
  • File → New: Extensive template collection
  • Categories: Business, Education, Design, etc.
  • Professional, time-saving starting points

Benefits of Using Templates

  • Consistent design throughout presentation
  • Professional appearance
  • Pre-designed layouts for different content types
  • Built-in color schemes and fonts
Demonstration: Browse template gallery and select a professional theme

Slides and Layouts

Building Blocks of Your Presentation

The Slide Interface

  • Thumbnail Pane: Left side - shows all slides
  • Main Editing Area: Center - edit current slide
  • Notes Section: Bottom - presenter notes

Working with Slides

Adding Slides

  • New Slide button: Uses current layout
  • New Slide dropdown: Choose specific layout
  • Keyboard shortcut: Ctrl + M

Essential Keyboard Shortcuts

  • Ctrl + D - Duplicate slide
  • Ctrl + M - New slide
  • Delete - Remove selected slide
  • F5 - Start slideshow from beginning
  • Shift + F5 - Start from current slide

Common Slide Layouts

Choosing the Right Layout for Your Content

Standard Layouts

  • Title Slide: Presentation title and subtitle
  • Title and Content: Most versatile layout
  • Two Content: Side-by-side comparisons
  • Comparison: Structured comparisons
  • Section Header: Dividing presentation sections

Layout Features

  • Placeholders: Pre-positioned content areas
  • Content icons: Quick access to media insertion
  • Consistent formatting: Automatic theme application
Right-click any slide → Layout to change the layout type
Practice: Create slides with different layouts and explore their uses

Adding Content to Slides

Text, Images, and Media

Text Content

  • Click in placeholder to add text
  • Use Insert → Text Box for custom placement
  • Format using the ribbon tools

Images

Inserting Images

  • Click image icon in content placeholder
  • Insert → Pictures from file or online
  • Stock images available through PowerPoint

Image Editing

  • Crop tool: Focus on specific areas
  • Picture Format: Styles, effects, colors
  • Alternative text: Accessibility requirement
Accessibility Note: Always add alternative text descriptions for images (Right-click → Edit Alt Text)

Adding Videos and Media

Enhancing Presentations with Multimedia

Video Integration

Inserting Videos

  • Insert → Media → Video
  • Options: Stock videos, online videos, or files
  • Resize by dragging corners
  • Crop videos like images

Video Formatting

  • Video Format tab: Shapes, effects, styles
  • Playback settings: Auto-play options
  • Custom shapes (hold Ctrl for circles)
Shape Tip: Hold Ctrl while selecting shapes to create perfect circles, squares, or equilateral triangles
Demo: Insert a video, resize it, crop it, and change it to a circular shape

Slide Management and Views

Different Ways to Work with Your Presentation

View Options

Normal View

  • Default editing view
  • Thumbnail pane + main editing area
  • Notes section at bottom
  • Best for detailed slide editing

Slide Sorter View

  • View all slides as thumbnails
  • Perfect for reordering slides
  • Bulk operations (delete, duplicate multiple slides)
  • Overview of entire presentation flow

Helpful Features

  • Grid lines: View → Grid lines (alignment help)
  • Ruler: View → Ruler (precise positioning)
  • Zoom controls: Bottom right corner
Bulk Selection: In Slide Sorter, click and drag to select multiple slides, then use Ctrl+D to duplicate them all

Slide Master - Theme Customization

Creating Consistent, Professional Designs

What is Slide Master?

  • Controls the overall theme and layout templates
  • Make changes that apply to all slides
  • Access via View → Slide Master

Slide Master Capabilities

Global Changes

  • Font styles and sizes
  • Color schemes
  • Background elements
  • Logo placement

Custom Layouts

  • Insert Layout: Create new slide types
  • Placeholders: Content, text, media areas
  • Shapes and graphics: Background elements
  • Rename layouts: Right-click → Rename
Important: Always click "Close Master View" when finished to return to normal editing

Animations and Transitions

Adding Motion to Engage Your Audience

Animations

  • Purpose: Control when content appears
  • Appear: Simple, professional effect
  • Animation Pane: Manage timing and order
  • Start options: On click, after previous, with previous
Best Practice: "Appear" animation is professional and rarely distracting

Transitions

  • Purpose: Control how slides change
  • Timing options: On mouse click or automatic
  • Duration: Match to audio or content needs
  • Consistency: Use same transition throughout

Creating Progressive Reveals

  1. Select multiple objects (hold Shift)
  2. Apply "Appear" animation
  3. Adjust timing in Animation Pane
  4. Test in slideshow mode

Presenter View and Delivery

Professional Presentation Delivery

Understanding Presenter View

  • Dual-screen setup: Audience sees slides, you see presenter view
  • Your view includes: Current slide, next slide, notes, timer
  • Access: Slideshow → Present from Beginning/Current Slide

Presenter Notes

  • Add notes in Normal view (bottom section)
  • Visible only to you during presentation
  • Include key points, transitions, timing cues
  • Use bullet points for easy reading

Display Settings

  • Swap displays: If screens are reversed
  • Monitor selection: Choose primary display
  • Slideshow settings: Advanced options
Test First: Always preview in presenter view before your actual presentation

Audio Integration and Automation

Creating Self-Running Presentations

Adding Audio

  • Insert → Audio → Record Audio for narration
  • Playback settings: Set to "Automatically"
  • Timing coordination: Match slide transition timing to audio length

Automated Presentations

Setting Up Auto-Advance

  1. Transitions tab
  2. Uncheck "On Mouse Click"
  3. Check "After" and set time duration
  4. Apply to all relevant slides

Export as Video

  1. File → Export → Create a Video
  2. Choose resolution quality
  3. Select timing options
  4. Save as MP4 file
Use Cases: Kiosks, online learning, recorded presentations, social media content

Accessibility Best Practices

Making Presentations Inclusive for All

Accessibility Checker

  • File → Info → Check for Issues → Check Accessibility
  • Identifies missing alt text, low contrast, reading order issues
  • Provides specific recommendations for fixes

Essential Requirements

  • Alt text: Describe all images meaningfully
  • High contrast: Dark text on light backgrounds
  • Reading order: Logical content flow
  • Title slides: Mark for screen readers

Writing Good Alt Text

  • Be concise but descriptive
  • Focus on the image's purpose in context
  • Avoid "image of" or "picture of"
  • Don't auto-generate - write your own
Legal Note: Many institutions require accessible presentations for compliance with disability laws

Design Best Practices

Creating Professional, Engaging Presentations

Visual Design Principles

  • Consistency: Use templates and master slides
  • Color palette: 1 primary + 1 accent color
  • High contrast: Ensure readability
  • White space: Don't overcrowd slides
  • Font hierarchy: Clear heading/body distinction

Content Guidelines

  • 6x6 rule: Max 6 bullet points, 6 words each
  • One concept per slide
  • Progressive disclosure: Use animations thoughtfully
  • Quality images: High resolution, relevant
  • Backup plan: Test multimedia beforehand

Professional Tips

  • Save work regularly (Ctrl+S) or use auto-save via cloud storage
  • Preview in presenter view before delivery
  • Test all multimedia elements and timing
  • Have a backup plan for technical issues

Continuing Your PowerPoint Journey

Resources for Mastery

Learning Approach

  • Practice regularly: PowerPoint proficiency comes with experience
  • Start simple: Master basic features before advanced techniques
  • Experiment: Try new layouts, animations, and design elements
  • Seek feedback: Test presentations with colleagues or students

Recommended Learning Path

  1. Master basic slide creation and layouts
  2. Learn effective design principles
  3. Practice multimedia integration
  4. Explore advanced features (Master slides, automation)
  5. Focus on accessibility and inclusive design

Remember: Great presentations combine strong content with thoughtful design and confident delivery. PowerPoint is the tool, but your message is what matters most.
