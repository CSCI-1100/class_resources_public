When calculating team sizes, order of operations matters:

ğŸ› ï¸ Building Our Own Team Generator

Let's combine everything we've learned to create a simplified version of the team generator tool we used earlier!

import random ## Part 1

def create_teams(student_names, team_size=4):
    """
    Create random teams from a list of student names.
    This is our algorithm design in action!
    """
    # Step 1: Make a copy and shuffle (randomization)
    students = student_names.copy()
    random.shuffle(students)
    
    # Step 2: Create teams (decomposition)
    teams = []
    team_number = 1
    
    # Step 3: Group students (pattern recognition)
    for i in range(0, len(students), team_size):
        team = students[i:i + team_size]
        teams.append({
            'number': team_number,
            'members': team
        })
        team_number += 1
    
    return teams
